Architecture

Engineered
from the ground up.

◈
Grouped Query Attention

Dramatically reduces memory bandwidth with shared key-value heads, enabling longer contexts at lower cost.

⋈
RoPE Embeddings

Rotary position embeddings extend context gracefully beyond training length without loss of coherence.

⚑
Interleaved Vision Encoder

A lightweight SigLIP-based encoder integrates visual tokens directly into the language stream.
